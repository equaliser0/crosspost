## Description

crosspost is a local posting app, which on several platforms at the same time for you. So you dont need to to make e.g. Twitter-promo-post for your article, which you posted on other platforms anymore

### Milestones

- [] Twitter
- [] Hive
- [] Medium
- [] Scale

[crosspost](https://github.com/equaliser0/crosspost)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
